What is LabelForge?

LabelForge implements programmatic weak supervision for creating training labels from multiple noisy sources

Intuitive API

Define labeling functions with a simple @lf decorator. Express domain expertise in Python code with zero configuration overhead.

Advanced Analytics

Real-time monitoring of labeling function performance, conflict detection, and coverage analysis with detailed visualizations.

Enterprise Scale

Process millions of examples with distributed computing, optimized EM algorithms, and intelligent memory management.

Probabilistic Models

State-of-the-art EM algorithms automatically learn function accuracies and correlations to produce calibrated confidence scores.

Open Source

Apache 2.0 licensed with active community development, extensive documentation, and enterprise support available.

ML Integration

Seamless integration with PyTorch, TensorFlow, Hugging Face, and MLflow for end-to-end ML workflows.

Quick Installation

Get started with LabelForge in minutes

Install from PyPI

pip install labelforge

Or install from source

git clone https://github.com/sonishsivarajkumar/LabelForge.git
cd LabelForge
pip install -e .

Requirements: Python 3.8+, NumPy 1.20+, Pandas 1.3+, SciPy 1.7+, scikit-learn 1.0+

See It In Action

Simple examples to get you started

Text Classification

from labelforge import lf, LabelModel, apply_lfs

@lf(name="positive_sentiment")
def sentiment_positive(example):
    positive = ["excellent", "amazing", "love"]
    return 1 if any(w in example.text.lower() 
                   for w in positive) else 0

@lf(name="negative_sentiment") 
def sentiment_negative(example):
    negative = ["terrible", "awful", "hate"]
    return 1 if any(w in example.text.lower() 
                   for w in negative) else 0

Medical Text Analysis

@lf(name="diabetes_keywords")
def diabetes_mentions(example):
    keywords = ["diabetes", "diabetic", "glucose"]
    return 1 if any(k in example.text.lower() 
                   for k in keywords) else 0

@lf(name="diabetes_symptoms")
def diabetes_symptoms(example):
    symptoms = ["insulin", "blood sugar", "neuropathy"]
    return 1 if any(s in example.text.lower() 
                   for s in symptoms) else 0

@lf(name="negative_diabetes")
def no_diabetes(example):
    negations = ["no diabetes", "not diabetic"]
    return 0 if any(n in example.text.lower() 
                   for n in negations) else -1  # abstain
